About MovieVerse

🎬 Welcome to MovieVerse

MovieVerse is a movie discovery web application that allows users to search for movies using the OMDb API, view detailed information, rate movies, and save their favorites.

The project is built using HTML, CSS, and JavaScript, with Local Storage used to save ratings and favorite movies.

✨ Features

  • 🔍 Search movies using the OMDb API
  • ⭐ Rate your favorite movies
  • ❤️ Save movies to your Favorites
  • 🏆 View your Top Rated movies
  • 🎬 View detailed movie information
  • 💾 Data saved using Local Storage
